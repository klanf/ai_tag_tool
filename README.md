# 🎨 AI 绘图标签分类工具

一个用于自动识别并分类 Danbooru 风格 AI 绘图标签的纯前端工具，支持分类复制、收藏管理、历史记录与随机标签组合生成。

## 📷 界面预览

![界面截图](screenshot.png)

## ✨ 功能特性

- **智能分类** — 将粘贴的标签自动归入 12 个语义类别（人物、服饰、表情动作、画面、环境、场景、物品、镜头、汉服、魔法系、ip、动作）
- **自动补全** — 输入框支持实时自动补全，支持中英文搜索
- **分类复制** — 一键复制单个分类下的所有标签，或复制完整分类结果、纯标签列表
- **收藏管理** — 标签可收藏到本地（localStorage），随时从收藏库快速添加到结果中
- **历史记录** — 分类结果保存到浏览器 localStorage，支持加载、删除历史快照
- **随机组合** — 基于历史记录或词典数据，按各类别的配额随机生成一组完整标签，快速获得灵感
- **搜索历史** — 添加标签弹窗支持搜索历史记录，可一键应用或删除
- **剪贴板联动** — 页面加载时自动检测剪贴板内容，提示是否直接载入
- **统计面板** — 实时展示总标签数、已知 / 未知数量、涉及分类数

## 🚀 快速开始

### 环境要求

- Python 3.x（仅后端 `server.py` 需要，无需第三方依赖）

#### 安装 Python

1. 访问 [Python 官网](https://www.python.org/downloads/) 下载最新版本
2. 安装时勾选 "Add Python to PATH"
3. 验证安装：

```bash
python --version
```

如果显示类似 `Python 3.12.x` 则安装成功。

### 启动方式

**Windows：** 双击 `start.bat`

**其他系统 / 手动启动：**

```bash
cd ai_tag_tool
python server.py
```

**停止服务：** 双击 `stop.bat`

启动后浏览器访问：

```
http://localhost:8080/index.html
```

> ⚠️ 必须通过 `server.py` 启动（而非直接双击打开 HTML），否则 `fetch` 加载 JSON 词典会因 `file://` 协议跨域限制而失败。

## 🏷️ 分类体系

| 类别键      | 显示名称 | 说明                         |
| ----------- | -------- | ---------------------------- |
| `1`         | 人物     | 角色主体（1girl、1boy、solo 等） |
| `2`         | 服饰     | 服装、配饰（dress、shirt、panties 等） |
| `3`         | 表情动作 | 姿势、表情、动作（blush、sitting、standing 等） |
| `4`         | 画面     | 画面效果、风格（style、quality、lighting 等） |
| `5`         | 环境     | 背景、环境（indoors、outdoors、sky 等） |
| `6`         | 场景     | 具体场景（classroom、bedroom、beach 等） |
| `7`         | 物品     | 道具、物品（sword、book、flower 等） |
| `8`         | 镜头     | 视角、构图（close-up、full_body、pov 等） |
| `9`         | 汉服     | 汉服相关标签 |
| `10`        | 魔法系   | 魔法、奇幻相关标签 |
| `11`        | ip       | 角色/IP名称 |
| `12`        | 动作     | 特殊动作标签 |
| `0`         | 未识别   | 无法归类的标签 |

## 🛠️ 技术栈

- **前端**：原生 HTML / CSS / JavaScript（单文件，无构建依赖）
- **后端**：Python `http.server`（零第三方依赖，仅提供静态文件服务）
- **存储**：`localStorage`（收藏 + 历史记录 + 搜索历史）

## 📖 使用流程

1. 启动服务器，浏览器打开页面
2. 将 Danbooru 标签粘贴到左侧输入框（逗号或换行分隔）
3. 点击「解析分类」，右侧按类别展示结果
4. 点击分类旁的「复制」按钮复制该类全部标签
5. 点击标签左侧的 ☆ 收藏常用标签
6. 可保存当前结果到历史、或点击「随机组合」获取灵感
7. 点击「+ 添加标签」弹窗可搜索和添加标签
